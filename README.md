## Технологии:
`HTML` `Scss` `JavaScript` `Gulp` `Webpack`

## Требования
Десктоп, мобильные устройства, современные версии браузеров

## Установка

```
$ npm i
```


## Запуск

```
$ npm run start 
```
Запускается на: `http://localhost:3000/`
